# AI-Based ATS Resume Filtering System

This project is an AI-based Applicant Tracking System (ATS) that compares resumes against job descriptions using NLP techniques.

## Features
- Domain & Role based job description selection
- Resume-to-job-description matching
- ATS score with PASS/FAIL logic
- Dashboard visualization using Streamlit
- Encrypted resume storage
- PDF & DOCX resume support

## Tech Stack
- Python
- Streamlit
- NLP (TF-IDF, Cosine Similarity)
- Plotly
- Cryptography

## How to Run
```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
